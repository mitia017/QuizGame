## 2025-05-14 - [Console Quiz UX Polish]
**Learning:** In console-based applications, providing progress tracking (e.g., "Question X/Y") and clear input prompts (e.g., "> ") significantly improves the user's sense of place and readiness to interact. Aggressive feedback (e.g., "Bad answer !!!") can be off-putting; neutral but clear feedback is preferred.
**Action:** Always include progress indicators and input prompts in CLI tools.
